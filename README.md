# longform-essay-writer

A Claude skill for writing captivating longform intellectual essays. Distilled from Zinsser's *On Writing Well* and Pinker's *The Sense of Style*.

Built for professional-audience pieces on history, geopolitics, and systems thinking — the kind that change how readers see something, not just inform them.

---

## What it does

Installs a set of principles Claude applies whenever you're writing or editing a longform essay:

- **Leads and hooks** — what makes a first sentence pull, and what kills it dead
- **Stripping clutter** — the bracket test, zombie nouns, redundant adverbs, qualifiers that telegraph timidity
- **Voice and clarity** — classic style as a window onto the world, not a performance
- **Organizing complex arguments** — topic strings, given-before-new, four structural templates
- **Why prose lands cognitively** — the mechanics behind parallel syntax, right-branching sentences, concrete imagery
- **Endings** — why summarizing is a failure, and what works instead
- **Intellectual essays specifically** — interrogating conventional wisdom, false balance, using history as argument not decoration

The central question the skill forces before every piece: *What mental model does the reader arrive with? What mental model should they leave with?*

## Usage

### With the `.skill` file (Claude.ai)

Download `SKILL.md`, install it in your Claude skills folder. Claude will apply the principles automatically when you ask it to write or edit an essay.

### Manually

Copy `SKILL.md` into any Claude conversation as context, then ask Claude to write or edit your piece with it.

---

## Pairing with the Humanizer

This skill handles **architecture** — structure, argument, lead, ending, voice.

The [humanizer skill](https://github.com/blader/humanizer) handles **surface** — stripping AI-isms after the draft is structurally sound.

Use both. A well-structured essay that still reads like it was generated by a language model hasn't cleared the bar.

Workflow:
1. Write or edit with `longform-essay-writer`
2. Polish with `humanizer`
